When Visual Studio creates a .NET MAUI project, four important code files are generated.

These can be seen in the Solution Explorer pane of Visual Studio


These files help get the .NET MAUI app configured(形成する構成する) and running(実行する).
これらのファイルは、,NET MAUI アプリを構成して実行するのに役立ちます。

Each(各々) file serves(仕える、奉仕する) a different purpose(目的), described(説明する) below(下の):

・MauiProgram.cs

this is a code file that bootstrap your app.

the code in this file serves as the cross-platform entry point of the app, which configures and starts the app.

The template startup code points to(指し示す) the App class defined by the App.xaml file.

・App xaml and App.xaml.cs

both of these files are referred(言及する、参照する) to as a single(単一の) file.

of in the Solution Explorer.
The xaml file contains XAML markup and the code file contains code created by the user to interact with
the XAML markup.
